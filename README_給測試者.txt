股票翻譯機 測試版

使用方式
1. 解壓縮整個 zip 到你喜歡的位置，例如桌面或文件資料夾。
2. 雙擊「股票翻譯機.exe」。
3. 瀏覽器會自動開啟 http://127.0.0.1:8765。
4. 搜尋股票代號或名稱，看數字、估價、雷達中心、K 線圖和本地資料日期有沒有怪怪的。
5. 測完可以直接關掉黑色視窗。

資料會放在哪裡
- 新版會把使用者資料放在：
  %LOCALAPPDATA%\StockTranslator\data
- 第一次開新版時，如果舊版 exe 旁邊的 data 資料夾有 stock_translator.sqlite3，程式會自動複製到新位置。
- 複製成功後不會刪掉舊 data；舊資料夾可以當備份。
- 之後更新程式只換 exe 與 _internal，不會覆蓋你的資料庫。

更新方式
1. 程式啟動後會自動檢查 GitHub Releases 是否有新版。
2. 若有新版，畫面上方會出現「下載並更新 / 直接下載 / 查看說明」。
3. 「下載並更新」會下載新版 zip，等程式關閉後備份舊版、換上新版，再重開。
4. 「直接下載」永遠保留。若防毒或 Windows 阻擋自動換版，請用直接下載手動解壓並覆蓋程式檔。
5. 自動檢查更新可以在右上角關閉；它只連 GitHub 取得版本號與下載連結，不上傳任何本地資料。

未簽章限制
- 測試版尚未購買程式碼簽章憑證。
- Windows SmartScreen 或 Defender 可能對 exe、下載的 zip、或自動更新用的 updater.bat 顯示警告。
- 如果公司或受控電腦不允許自動替換，請改用「直接下載」手動更新。

請幫忙測
- 首次升級：舊版旁邊有 data 時，新版是否會自動接上資料。
- 本地資料：日線、法人、雷達快照的日期是否合理。
- 更新通知：有新版時是否看得到「直接下載」備援。
- 自動更新：按「下載並更新」後是否能重開新版；若被防毒攔截，請截圖錯誤。
- 個股頁：同步、估值、K 線、法人卡片與報告是否正常。

請幫忙回報
- 哪一檔股票代號。
- 哪個數字或日期看起來怪，例如股利、估值情境、漲跌、資料日期、雷達狀態。
- 你期待看到什麼數字，或你拿哪個 App / Excel 對照。
- 截圖整個畫面。

注意
- 這是本機測試版，不會自動下單。
- 內容只是在整理與翻譯資料，不是投資建議。
- 若要回報完整資料問題，可以把 %LOCALAPPDATA%\StockTranslator\data\stock_translator.sqlite3 一起傳回來。
