股票翻譯機 測試版

使用方式
1. 解壓縮整個 zip。
2. 雙擊「股票翻譯機.exe」。
3. 瀏覽器會自動開啟 http://127.0.0.1:8765。
4. 搜尋股票代號或名稱，看數字、估價、雷達中心、K 線圖和本地資料日期有沒有怪怪的。
5. 測完可以直接關掉黑色視窗。

今日更新功能（2026-06-25）
1. 大型 K 線圖新增「盤中暫算」：只在大型圖用盤中成交價疊加暫算 K 棒，不寫入正式日線資料。
2. 盤中支撐壓力：開啟暫算後，支撐/壓力線會用含盤中 K 棒的畫面資料重新計算，並標示「暫算 / 盤中」。
3. 收盤資料隔離：小型 K 線圖、研究報告、匯出、估值與正式資料仍維持收盤日線；若正式日線已含該日期，畫面會顯示不另疊加。
4. 讀圖導覽：大型 K 線圖新增導覽播放器，會依序高亮均線、位階、關卡、動能、量能、事件、籌碼與觀察關卡；全程只做讀圖識讀，不給買賣訊號。
5. 關係層：導覽新增「這檔好不好讀」與「這次漲跌扎不扎實」，會把價、量、法人是否同向整理成白話，並用結構指紋做可信度折扣。

前次更新功能（2026-06-24）
1. 大型 K 線圖：新增大型圖模式，把進階指標、圖層控制、完整資料室集中到大型 K 線圖；一般小型 K 線圖維持簡潔翻譯版。
2. K 線翻譯：新增圖表翻譯摘要、天氣標籤、情境區間與指標說明，幫助理解目前圖形，不是預測。
3. 個股結構指紋：新增延續性、複雜度、波動聚集、噪音色、湍流程度等描述卡，說明這檔股票近期行為型態。
4. 市場心智雷達：雷達中心新增市場層級分頁，描述市場分化、羊群程度、同步度；資料不足時會引導先去本地資料下載。
5. 本地資料日期修正：今天 2026-06-24 檢查時，最近完整收盤目標會抓 2026-06-23，不會再把 2026-06-22 誤判成最新。
6. 詞彙與安全文案：新增市場雷達詞彙說明，所有結構/市場指標都維持「描述現在、不預測、不給買賣訊號」。

請幫忙測
- 大型 K 線圖：按「大型K線圖」後開啟「盤中暫算」，確認最後一根 K 棒與支撐壓力標籤有明確暫算提示。
- 讀圖導覽：按「讀圖導覽」後確認能上一步/下一步/暫停，圖上高亮是藍、金、灰，不應出現預測、目標價或買賣指令。
- 關係層：導覽前段應先說這檔好不好讀，再說最近漲跌有沒有量與大戶同步；文字只能描述扎實度，不應變成方向判斷。
- 小型 K 線圖：不要出現盤中暫算控制或巨型指標區，仍維持簡潔翻譯版。
- 雷達中心：市場心智雷達是否能載入；資料不足時是否有清楚引導。
- 個股頁：小型 K 線圖是否仍維持簡潔；按「大型K線圖」後進階功能是否集中在大型圖。
- 本地資料：6/24 測試時，若最後資料日是 6/22，狀態應顯示缺 1 日或快照待更新，而不是「日線已最新」。
- 同步：按同步後應該會嘗試補到最近完整收盤日；若資料源還沒發布，請回報畫面訊息。
- 結構指紋：文字應該只描述目前型態，不應出現預測、目標價或操作建議。

請幫忙回報
- 哪一檔股票代號。
- 哪個數字或日期看起來怪，例如股利、估值情境、漲跌、資料日期、雷達狀態。
- 你期待看到什麼數字，或你拿哪個 App / Excel 對照。
- 截圖整個畫面。

注意
- 這是本機測試版，不會自動下單。
- 內容只是在整理與翻譯資料，不是投資建議。
- data 資料夾裡的 stock_translator.sqlite3 是測試資料庫；如果要回報完整問題，可以把這個檔案一起傳回來。
