股票翻譯機 測試版

使用方式
1. 解壓縮整個 zip。
2. 雙擊「股票翻譯機.exe」。
3. 瀏覽器會自動開啟 http://127.0.0.1:8765。
4. 搜尋股票代號或名稱，看數字、估價、雷達中心有沒有怪怪的。
5. 測完可以直接關掉黑色視窗。

請幫忙回報
- 哪一檔股票代號。
- 哪個數字看起來怪，例如股利、便宜價、合理價、昂貴價、漲跌、日期。
- 你期待看到什麼數字，或你拿哪個 App / Excel 對照。
- 截圖整個畫面。

注意
- 這是本機測試版，不會自動下單。
- 內容只是在整理與翻譯資料，不是投資建議。
- data 資料夾裡的 stock_translator.sqlite3 是測試資料庫；如果要回報完整問題，可以把這個檔案一起傳回來。
