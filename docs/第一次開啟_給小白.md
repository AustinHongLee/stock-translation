# 第一次開啟教學（看到藍色畫面別怕）

> 這支程式是免費工具、沒有付費的「數位簽章」，所以 Windows 第一次會先攔一下，**這是正常的**。
> 它只會連台灣證交所抓公開股價，**不會上傳你電腦裡的任何資料**。照下面做一次就好，之後更新不會再跳。

---

## 步驟 1：整包解壓縮
把下載的 zip **整個資料夾解壓縮**出來（不要只把 `股票翻譯機.exe` 單獨拉出來，旁邊的 `_internal` 要一起）。

## 步驟 2：雙擊「股票翻譯機.exe」

## 步驟 3：出現藍色畫面時，這樣點（重點！）
畫面標題是「**Windows 已保護您的電腦**」。
1. 它一開始**只會有一顆「不要執行」**——先別關掉。
2. 點畫面中間那行小小的「**更多資訊**」。
   〔截圖位置：用紅圈圈住「更多資訊」這行字〕
3. 這時下面才會多出「**仍要執行**」按鈕 → 按它。
   〔截圖位置：用紅圈圈住「仍要執行」按鈕〕

✅ 程式就會打開了。**以後再開、或程式自己更新，都不會再跳這個畫面。**

---

## 可能遇到的兩個特例

### A. 畫面寫「已封鎖」而且**找不到「仍要執行」**
這是較新 Windows 11 的進階防護「**Smart App Control**」（多數電腦沒開）。要跑的話：
- 開「**Windows 安全性**」→「**應用程式與瀏覽器控制**」→「**Smart App Control**」→ 設為「**關閉**」，再重開程式。
- （不想關防護的話，就先用別台沒開這功能的電腦。）

### B. 防毒把 exe「隔離／刪除」了
少數防毒會把沒簽章的新程式當可疑。到你的防毒軟體裡找到隔離區，選「**允許／還原**」這個檔即可。

## 你的資料放哪、會不會不見？
- 下載的股價／法人資料存在：`C:\Users\你的名字\AppData\Local\StockTranslator\data`
- **程式更新不會碰到這個資料夾**，所以更新後資料都還在，不用每次重抓。

有問題就把畫面截圖傳給我們，我們幫你看。
