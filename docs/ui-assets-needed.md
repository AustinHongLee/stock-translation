## 實作備註

- HTML 已用 `<img loading="lazy" alt="…">` 引用上述路徑，並包在帶 CSS 底色的容器內。
- 若圖片載入失敗，`onerror` 會把 `img` 隱藏、露出容器的 CSS placeholder 底色，**不會出現破圖 icon**，UI 不被阻塞。
- 換正式圖時：直接覆蓋同名檔案即可，不需改 HTML/CSS。
- `app/ui/static/assets/placeholders/` 目錄保留給日後其他暫時素材（目前為空）。
